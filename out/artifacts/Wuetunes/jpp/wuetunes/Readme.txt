Readme zu Wuetunes:
-Erstellen einer Playlist per Drag and Drop -> Sobald ein Song in der Playlist ist wird der Mediaplayer angezeigt
-Sortieren der Playlist durch ziehen eines Songs an das Ende der Playlist.
-Löschen eines Songs per Drag und Drop -> Ziehen des ausgewählten Songs zurück in die Rechte Tabelle-> Er wird aus der Playlist gelöscht.
-Setzen eines Ratings durch Doppelklick auf die Rating Zelle des entsprechenden Songs -> Auswahlbox mit Rating Möglichkeiten erscheint.
-Filter: Eingabe des gewünschten Filters -> Drücken auf Apply Filter -> Filter werden angewandt
        -> Um erneut alle Musikstücke anzuzeigen alle Textfelder leeren und erneut den Apply Filter Button betätigen
